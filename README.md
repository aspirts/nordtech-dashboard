# Datu analītikas projekts: NordTech Dashboard

## Projekta apraksts:
Šis datu panelis ir izstrādāts, lai vizualizētu un analizētu uzņēmuma operatīvos datus. Projekta ietvaros tika veikta datu tīrīšana, apstrāde un interaktīvu vizualizāciju izveide, izmantojot **Python** bibliotēkas.

### Galvenās funkcijas:

- **Datu integrācija:** Automātiska datu ielāde no dažādiem avotiem (CSV, Excel, JSONL).
- **Interaktīva vizualizācija:** Dinamisku grafiku izveide, izmantojot **Plotly** bibliotēku.
- **Mākoņdatošana:** Lietotnes publicēšana, izmantojot **Streamlit Community Cloud** un **GitHub** integrāciju.

### Tehnoloģiskais steks:

- **Python** (Pandas, Plotly Express, Streamlit).
- **GitHub** versiju kontrolei.
- **Streamlit Cloud** hostingam.
